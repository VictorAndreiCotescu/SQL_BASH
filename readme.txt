UPDATE

	Name
	
		UPDATE - Modifica valori in baze de date

	Descriere

		Functia primeste ca argumente table-ul in care urmeaza sa 
		modifice valorile, coloanele ce trebuie modificate cu valorile
		noi si conditiile pentru realizarea acestei modificari.

	Synopsis

		UPDATE [TABLE NAME] SET [COLUMN NAME] = [NEW VAL] .. WHERE [CONDITIONAL COL] = [CONDITIONAL VAL} ..

	Author

		Victor Andrei Cotescu
